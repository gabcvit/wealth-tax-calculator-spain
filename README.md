# Wealth Tax Calculator — Spain 🇪🇸

A simple and accurate Wealth Tax (Impuesto sobre el Patrimonio) calculator for **Spain**, compatible with:

✅ Spanish **residents**  
✅ **Non-residents**  
✅ Scenarios with **multiple buyers/co-owners**  
✅ Assets with **outstanding loans / mortgages**

This tool helps you estimate your Spanish wealth tax liability based on real rules and exemptions.

---

## 📌 Features

- 🧮 Calculates total taxable wealth taking into account:
  - Property (real estate)
  - Financial assets (investments, cash, securities)
  - Other taxable assets per Spanish tax law
- 📊 Adjusts values for **loans and mortgages**
- 👥 Supports any number of buyers/co-owners — tax share adjusts accordingly
- 🏠 Applies relevant **regional allowances** (Autonomous Community minimums)
- 🇪🇸 Resident vs Non-Resident treatment
- 📅 Fiscal year aware

---

## 🧠 How It Works

Wealth tax in Spain is a **progressive tax** applied to net assets above regional minimum thresholds. This calculator:

1. Aggregates all assets
2. Deducts valid debts (like outstanding mortgages)
3. Applies regional and national exemptions
4. Splits taxable base among co-owners
5. Computes wealth tax using applicable tax brackets
